# Simon Game

An interactive browser-based memory game inspired by the classic Simon Game.  
The application challenges users to remember and repeat an increasingly complex sequence of button presses.

## How the Game Works

1. Press any key to start the game.
2. A random button flashes.
3. Repeat the sequence in the correct order.
4. Each level adds a new color to the sequence.
5. The game ends if the wrong button is pressed.
